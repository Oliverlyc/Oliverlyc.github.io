linux添加环境变量

在Linux系统下,安装完应用程序之后,输入应用名称会提示”command not found“,为了避免每次都要打开文件夹执行程序,引入了环境变量,将路径添加到$PATH中

  1. # vim ~/.bashrc
  2. 在最后一行加入
    # PATH
    export PATH=路径:$PATH
  3. 保存退出执行命令
    # source ~/.bashrc
